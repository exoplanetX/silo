## 7. Diagnostics Dataclasses

Create or update:

```text
src/silo/presolve/diagnostics.py
```

Define a status enum or string constants for presolve status.

Recommended enum:

```python
class PresolveStatus(str, Enum):
    NOT_RUN = "not_run"
    NO_CHANGE = "no_change"
    REDUCED = "reduced"
    INFEASIBLE = "infeasible"
    UNBOUNDED = "unbounded"
```

Define:

```python
@dataclass(frozen=True)
class PresolveWarning:
    code: str
    message: str
    source: str | None = None
```

Define:

```python
@dataclass(frozen=True)
class PresolveDiagnostics:
    status: PresolveStatus = PresolveStatus.NOT_RUN
    warnings: tuple[PresolveWarning, ...] = ()
    removed_rows: tuple[str, ...] = ()
    removed_variables: tuple[str, ...] = ()
    fixed_variables: tuple[str, ...] = ()
    notes: tuple[str, ...] = ()
```

You may adjust field names if better, but include:

```text
status
warnings
removed rows
removed variables
fixed variables
notes
```

The dataclasses should be immutable and deterministic.

Do not put mutable lists as defaults.

## 9. Reduction Records

Create or update:

```text
src/silo/presolve/reductions.py
```

Define a small reduction-type enum or string constants.

Recommended enum:

```python
class ReductionType(str, Enum):
    EMPTY_ROW = "empty_row"
    EMPTY_COLUMN = "empty_column"
    FIXED_VARIABLE = "fixed_variable"
    BOUND_TIGHTENING = "bound_tightening"
    SCALING_DIAGNOSTIC = "scaling_diagnostic"
    NO_OP = "no_op"
```

Define:

```python
@dataclass(frozen=True)
class ReductionRecord:
    reduction_type: ReductionType
    target: str
    description: str
    data: Mapping[str, object] = field(default_factory=dict)
```

Important requirement:

`data` should not expose a mutable dictionary that can be modified externally.

Suggested implementation:

```python
from types import MappingProxyType
```

or convert to an immutable representation such as:

```python
tuple[tuple[str, object], ...]
```

If using `MappingProxyType`, consider whether dataclass equality remains intuitive. Simpler is acceptable:

```python
data: tuple[tuple[str, object], ...] = ()
```

Add helper:

```python
def reduction_data(**items: object) -> tuple[tuple[str, object], ...]:
    ...
```

The helper should return items sorted by key for deterministic order.

Do not implement actual reduction application or recovery logic yet.
